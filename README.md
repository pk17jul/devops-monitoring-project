# 🚀 DevOps Monitoring Project

A complete end-to-end DevOps project that monitors server health and deploys using modern DevOps tools.

## 📌 Features
- CPU, Memory, Disk monitoring
- Multi-server health checks (Ping)
- Web dashboard (Flask)
- Docker containerization
- CI/CD pipeline (Jenkins)
- Infrastructure as Code (Terraform)
- Kubernetes deployment (Coming next)

## 🛠️ Tech Stack
Python, Flask, Linux, Git, Docker, Jenkins, Terraform, AWS, Kubernetes
